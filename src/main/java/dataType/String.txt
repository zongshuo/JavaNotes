String:
    1、不能被继承
        A、被final关键字修饰
final：
    1、修饰的类不能被继承
    2、修饰的方法不能被重写
    3、修饰的变量，如果是基本数据类型则初始化后不能被修改。
        如果是引用类型则初始化后不能修改引用地址。
transient：被该关键字标注的变量不会被序列化。
String为什么是final的？
    首先String的部分实用特性是不能改变的，重写会破坏该特性，所以使用final拒绝继承。
    不可改变是内存地址的内容不变，两次赋值指向不同的地址。
    其次是为了安全。
    最后是为了降低消耗。

接口中的变量默认为：public static final 且必须赋初值。
